Namespace Backend.Services

Classes

CompatibilityProxy

Реалізація патерна Proxy (Замісник). Забезпечує шар кешування для важких операцій аналізу сумісності інгредієнтів.

ConflictNotifier

Суб'єкт, який керує списком спостерігачів та розсилає їм повідомлення. Патерн Observer.

DiagnosticDecorator

Реалізація патерна Decorator для моніторингу продуктивності. Додає функціонал вимірювання часу виконання до будь-якої стратегії сумісності.

ProductBuilder

Реалізація патерна Builder для покрокового створення об'єктів косметичних засобів (Product).

ReportGenerator

Абстрактний базовий клас, що реалізує патерн Template Method для генерації звітів. Визначає скелет алгоритму формування звіту.

SecurityLogger

Конкретна реалізація спостерігача, яка записує сповіщення у системний лог.

SimpleCompatibilityStrategy

Реалізація конкретної стратегії перевірки сумісності. Реалізує патерн Strategy.

SimpleTextReport

Конкретна реалізація генератора звітів у текстовому форматі.

SkincareFacade

Надає спрощений інтерфейс для взаємодії зі складною системою аналізу складу. Використовує патерн Facade.

SystemSettings

Глобальні налаштування системи. Реалізує патерн Singleton для забезпечення єдиної точки доступу до конфігурації.

Interfaces

ICompatibilityStrategy

Інтерфейс, що визначає контракт для алгоритмів перевірки сумісності. Реалізує патерн Strategy.

IObserver

Визначає інтерфейс для об'єктів, які мають отримувати сповіщення від суб'єкта. Компонент патерну Observer.
